# Smart Productivity Todo Manager

A modern productivity-focused Todo web application designed for students, professionals, and developers.

## Features

- Task management with priorities (Low, Medium, High)
- Deadline scheduling with date and time
- Task filtering (All, Active, Completed)
- Light/Dark theme support
- Clean, responsive UI with colorful cards
- Priority tags and visual indicators
